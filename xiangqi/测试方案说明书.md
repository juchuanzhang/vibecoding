## 4. 测试执行

### 4.1 自动化测试

```bash
cd engine
pytest -v                  # 运行所有单元测试
pytest tests/test_opening.py  # 棋谱测试
```

### 4.2 手动测试检查清单

```
□ 1. 启动 server.py，浏览器打开 localhost:8080
□ 2. 棋盘显示标准开局布局
□ 3. 走一步棋，棋盘实时更新
□ 4. 走非法棋（蹩马腿），棋盘不变，显示错误提示
□ 5. 双方交替走10步
□ 6. 点击悔棋，棋盘回退
□ 7. 点击分析，先看到浅层结果，再看到深层结果
□ 8. 分析期间UI完全可用（可走棋、点按钮）
□ 9. 切换搜索深度到8层，再次分析
□ 10. 导出FEN → 新局 → 导入FEN → 棋局恢复
□ 11. CPU使用率在分析期间达到多核占用
□ 12. 走棋历史中文描述正确（炮二平五、马8进7等）
□ 13. 评分显示"红方领先/黑方领先"文字
□ 14. Top 3 候选箭头不同颜色
